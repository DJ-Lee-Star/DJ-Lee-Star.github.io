[S8] 심사 정보 자동입력(A8/AG‑03) · Full Light
검증 완료 항목 자동입력 → WINI 반영(xFrame) → 상태 저장
검증 완료된 데이터 중 자동입력 가능한 항목을 WINI2(TBD) 심사 전자서식에 자동 입력하고, 입력 결과를 WINI에 반영(xFrame)하여 후속 심사 업무로 연계합니다.
Animation 필드 순차 채움 + 저신뢰 항목은 “확인필요” 유지

자동입력 대상 항목

검증 완료(일치) 항목만 자동 입력되고, 저신뢰/미대상은 확인 필요로 남습니다.

A8 Auto Fill
Tip 자동입력 실행을 누르면 “자동입력 가능” 항목만 순차 입력됩니다.

심사 전자서식(입력 화면)

자동입력은 검증 완료 항목만 적용되며, 확인 필요 항목은 담당자가 수정(HITL)합니다.

포인트: 자동입력은 “검증 완료(일치)” 기반으로만 수행되고, 예외/저신뢰는 자동입력되지 않습니다.

입력 결과/상태

자동입력 완료 → WINI 반영(xFrame) → 상태 저장 흐름을 로그로 보여줍니다.
