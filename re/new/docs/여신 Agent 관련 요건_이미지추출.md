# 여신 특화 Agent

## 주요 내용

- 대화형 인터페이스 기반 여신 업무 지원 목적 Agent 구현
  - 대화형 지능 및 지식 기반 답변(QA) 기능 제공
  - 기업여신 과제 내 Agent 처리 결과에 대한 검증, Action Plan 제언, Agent 처리 결과에 대한 지식 기반 답변(QA) 기능 제공
  - 공통 서비스 화면을 통한 통합단말(WINI)에 값 입력 지원
  - 고객 서류 파일 업로드를 통한 서류식별 및 진위확인(RPA 연동) 지원
  - 전자팩스시스템/이메일 등 연계를 통한 서류 식별 및 진위확인(RPA 연동) 지원

## 개발요건 상세

### 데이터 요건

#### 가) 여신 관련 지식/문의 내용 자산화

- RAG 검색 필요 데이터(내외 규정, 시행문, 공문, 업무매뉴얼 등) 식별 및 자산화
- 여신문의(전화/메신저, 게시판 등) 비정형 데이터 자산화(여신인력 투입으로, 여신관련 데이터 전처리 지원 예정)
- 여신 Agent 구현을 위한 필요 데이터 식별 및 자산화
- 행내 KPI 데이터 자산화 고도화
- 데이터 최신화 및 정합성 검증 자동화 프로세스 구축

### 여신업무 E2E 지원 대화형 인터페이스 Agent 구현

#### 나) 여신 신청 및 심사

- 사용자 문의에 대한 답변 제공(QA) 및 Action Plan 제언
- 징구 서류 OCR 처리 결과 中 기준치 미달 건에 대한 수정사항 제언
- OCR 처리 결과 및 수집 정보 기반 징구 서류 문의에 대한 답변(QA)
- 상품별 특성에 따른 필요 서류 제언
- 보증서 특약 분석 후 사용자 Action Plan 제언
- 관련 데이터 기반 공유주주 누락 점검, Action Plan 제언

- 기타 사용자 요청에 대한 처리

#### 다) 여신 실행

- 사용자 문의에 대한 답변 제공(QA) 및 Action Plan 제언
- 누락 서류 발생 시 알림 및 누락 서류 안내
- 징구 서류 內 필수 기업 부분 누락내용 식별 및 사용자 안내

#### 라) (사후관리) 여신 기일 관리 / 차주변동정보 확인 자동화

- 사용자 문의에 대한 답변 제공(QA) 및 Action Plan 제언
- 사용자별 여신만기 도래 기일 관리 필요 항목 도출
- 계정계 데이터 실시간 연계로 사용자 업무처리에 따른 결과값 반영 필요
- 차주변동정보 발생 시 기일 관리 화면 內 도출 및 Action Plan 제언
- 동정정보 요약보고서 확인 및 연계 시스템 전달

### 시스템 연계

- Legacy 시스템 (통합단말) 연계
- 거래화면 별 지식상담(QA) 및 Action Plan 등을 제언하기 위한 시나리오 구성 필수

### 화면 요건

#### 가) 공통서비스 화면 內 화면 구현 필요.

## 기타 고려사항

- 개인정보보호법 및 당행 규정 준수
- 서비스 가능 수준 Hybrid DB(GraphDB, RDB, VectorDB) 설계 및 최적화

# 여신 서류 입력 및 검수 자동화

## 주요 내용

- AI-OCR 기반 여신 관련 서류 적정 여부 검증
- 고객 제출 서류의 분류 및 적정성 검증, 서류 진위확인 값 식별 후 통합단말 입력
- 실물 제출 외 파일화 된 서류의 BPR W/F시스템 자동 전송
- 고객 작성 약정 서류의 적정 작성 여부 검증 후 오류 검출 및 안내

## 개발요건 상세

### 서류 등록/검증/추출/입력 등 프로세스

- 공통서비스과 W/F시스템, 이메일시스템, 전자팩스시스템 연계 등 파일업로드를 통한 서류 제출 진행
- 고객 제출 서류의 종류 구분
- 고객 제출 서류의 진위 여부 확인
- 고객 제출 서류 내 주요 값 추출
- 추출된 값의 통합단말 입력 전 공통서비스를 통해 서류이미지와 통합단말 입력 값의 하이라이팅 등의 표기를 통한 입력 검토

### 제출 서류의 종류

- 사업자등록증, 법인등기사항전부증명서, 소득서류, 자금목적서류(분양/매매/임대/기계설비 구매 계약서 등), 부동산 신고필증, 외화지급보증서, 인감증명서, 주민등록등초본, 부가가치세 과세표준증명, 매입매출처별 세금계산서 합계표, 종합소득세 과세표준확정신고서, 부동산임대공급가액 명세서, 착공신고서, 표준재무제표, 재무제표확인원 등 여신심사에 필요한 서류

### 업무 종류

- 부동산 관련 서류의 값 활용한 사정의견서(추정가액조사표) 등 담보평가 작성 지원
- 재무 관련 서류의 값 활용한 회계처리 기준에 따른 통합단말 입력 지원
- 소득 관련 서류의 값 활용한 통합단말 입력 지원
- 수기 입력 값 존재 시 서류 추출 값과 대사를 통한 오입력 알림 및 추가 검증
- 주요 고객 작성 서류(약정서 등)의 인감/서명 날인 등 주요정보 누락 여부 알림
- 여신 자금목적 서류의 주요 값 추출을 통한 전결권 유의사항 안내
- 실물 보증서 서류의 특약사항과 당행 사고사례 대조를 통한 유의사항 안내
- 실제소유자 서류(주주명부 등)와 내부 데이터(KYC 자료 등) 교차 검증을 통한 공유차주 누락방지 등 유의사항 안내
- RPA 연계를 통한 자동화 업무 지원

## 기타 고려사항

- 성과수치 Factor 수집 및 중앙화를 통한 모니터링 개발 필요
- 서비스 가능한 수준의 동시 접속자, 속도 고려 필요
