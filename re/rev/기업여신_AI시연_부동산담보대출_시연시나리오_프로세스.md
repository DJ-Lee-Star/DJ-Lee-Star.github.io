# 기업여신 AI 시연 부동산담보대출 시나리오 및 프로세스

## 1. 문서 목적
본 문서는 우리은행 기업여신 AI 시연을 `부동산 담보 기업대출` 시나리오 기준으로 고정하기 위한 보조 설계 문서다.
RFP 요구사항 중 담보 관련 서류 활용, 담보평가 작성 지원, Action Plan 제언을 시연 구조에 어떻게 반영할지 정의한다.

---

## 2. 대표 시나리오
- 차주: `(주)000바이오`
- 상품: `기업일반자금대출`
- 담보유형: `부동산담보`
- 자금목적: `운전자금`
- 주요 이슈:
  - 법인등기부등본 자본금 불일치
  - 주주명부 미제출
  - 담보 관련 서류 중 `사정의견서(추정가액조사표)` 검토 필요

---

## 3. 시연용 주요 서류군

### 3.1 기본 서류
- 사업자등록증명원
- 법인등기부등본
- 재무제표증명
- 부가가치세 과세표준증명
- 주주명부

### 3.2 부동산 담보 서류
- 부동산 등기부등본
- 건축물대장 또는 토지대장
- 사정의견서 또는 추정가액조사표
- 매매계약서 또는 담보 제공 관련 계약서

---

## 4. 화면별 반영 포인트

## G1
- 담보유형을 `부동산담보`로 고정
- 필요 서류 산출 영역에 담보 서류를 포함
- AI 제언:
  - 부동산 담보 대출은 권리관계와 담보가치 확인 서류가 추가로 필요하다는 안내

## G2
- 담보 서류 등록 현황을 일반 서류와 구분 표시
- AI 제언:
  - 담보 서류 누락 시 분석은 진행 가능하지만 담보 검토 단계에서 보완 필요 가능성 안내

## G3
- Agent 처리 결과에 담보 관련 이상 징후 또는 검토 필요 메시지 포함
- AI 제언:
  - 담보물 권리관계 확인 필요
  - 담보가치 산정 근거 확인 필요
  - 감정/사정 의견서 기반 추가 조치안 제시

## G4
- 브릿지 알림 사유에 담보 검토 요청 포함 가능

## G5
- 브릿지 검토 항목에 일반 필드 외 담보 검토 카드 포함
- AI 제언:
  - 선순위 권리 존재 시 한도 재검토
  - 담보가치 불충분 시 추가 담보 또는 조건 조정 권고

## G6
- 담보 관련 서류를 근거로 담보 검토 결과를 확정
- AI 제언:
  - 한도 조정
  - 추가 서류 보완
  - 조건부 심사 진행

## G7
- 단말 반영 결과에 담보 검토 결과 반영 내역 표시

## G8
- 심사 요청 또는 보완 요청 분기 시 담보 관련 보완 사유 포함

---

## 5. 단계별 AI 제언 기준

### 5.1 서류 산출 단계 제언
- 상품/담보/자금목적 기준 필요 서류 제언
- 담보 서류 누락 위험 안내

### 5.2 Agent 처리 단계 제언
- OCR 수정 제안
- 자본금 불일치 수정 제안
- 주주명부/KYC 보완 제안
- 담보가치 검토 필요 제안

### 5.3 브릿지 검토 단계 제언
- 수정 반영 제안
- 보완 요청 제안
- 한도/조건 조정 검토 제안

### 5.4 최종 분기 단계 제언
- 심사 요청 가능 사유
- 보완 요청 필요 사유
- 후속 체크리스트

---

## 6. 결론
부동산 담보 대출을 대표 시나리오로 고정하면, RFP의 다음 요구사항을 시연에 자연스럽게 반영할 수 있다.
- 상품별 필요 서류 제언
- OCR 결과 기반 수정사항 제언
- 실제소유자/KYC 교차 검증
- 담보 관련 서류 활용과 담보평가 작성 지원
- Action Plan 제언

향후 구현은 본 문서를 기준으로 일반 기업대출 화면이 아니라 `부동산 담보 기업여신 심사 보조 화면`으로 구체화해야 한다.
