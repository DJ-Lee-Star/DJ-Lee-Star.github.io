# 기업여신 AI Agent 시연 — 기능명세서

> 본 문서는 `p1.html`~`p12.html` 및 `common.js`의 실제 구현을 기준으로 기능을 식별·정의하고,  
> RFP 미반영 요건을 수용하기 위한 **확장 기능**을 함께 명세한다.  
> `[현행]`은 현재 구현 기능, `[확장]`은 RFP 충족·보강을 위해 추가가 필요한 기능이다.

## 2. 공통 기능 (FN-C00)

### FN-C01 페이지 전환 `[현행]`

| 항목 | 내용 |
|------|------|
| **기능 ID** | FN-C01 |
| **기능명** | 페이지 간 fade-in/out 전환 |
| **소속 모듈** | `common.js → WBPageTransition` |
| **트리거** | 각 페이지의 데모 시퀀스 완료 시점 |
| **입력** | 대상 URL, 딜레이(ms) |
| **처리** | 1) opacity 1→0 (130ms) 2) location.href 변경 3) 신규 페이지 opacity 0→1 (130ms) |
| **출력** | 페이지 전환 완료 |
| **참조 데이터** | 없음 |
| **제약 사항** | MPA 구조로 인해 전체 DOM 리로드 발생, 상태 유실 |

### FN-C02 자동 재생 (Auto-Paging) `[현행]`

| 항목 | 내용 |
|------|------|
| **기능 ID** | FN-C02 |
| **기능명** | 12페이지 순차 자동 재생 |
| **소속 모듈** | `common.js → WBAutoPaging` |
| **트리거** | `IS_AUTO_PAGING = true` 상태에서 각 페이지 데모 완료 |
| **입력** | 시퀀스 배열 `["p1.html"~"p12.html"]`, 딜레이(ms) |
| **처리** | 현재 페이지명 추출 → 시퀀스에서 다음 페이지 결정 → FN-C01 호출 |
| **출력** | 다음 페이지로 자동 전환 |
| **제약 사항** | P12 이후 순환 없음, 역방향 이동 미지원, 중간 점프 미지원 |
